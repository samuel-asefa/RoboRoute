What is RoboRoute?

It is a simulation tool to help teams plan out their
auton and programming skills in a simplistic and
visual way!

Commands:

  • move X - Move forward X pixels.
  • back X - Move backward X pixels.
  • turn X - Turn X degrees clockwise.
  • turn -X - Turn X degrees counter clockwise.
  • score X - Adds X point(s) to the score.

Dimensions:

Field: 800 x 800 (144'' x 144'')

Tile: 100 x 100 (24'' x 24'')

1 inch = 5.56 pixels
